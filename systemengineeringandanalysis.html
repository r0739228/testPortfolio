System engineering & analysis

Februari 14, 2020 Geel

Intro

Aan dit project heb ik alleen maar super leuke herinneringen. Wij hebben een hele week heel hard gewerkt om tot dit leuke resultaat te komen. De opdracht kwam van een ouder man die als hobby duivenmelker was. Hij was niet meer zo goed te been en zou graag weten wanneer hij zou moeten opstaan om gaan te kijken of zijn duiven bijna terug thuis zijn van een wedstrijd.

Onze opdracht

Deze opdracht was een allround project. Wij moesten zowel het harnasje ontwerpen als ook een gemakkelijk te gebruiken interface voor deze man. Ik heb me persoonlijk heel veel bezig gehouden met het schrijven van de code die ervoor zorgde dat de gebruiker duiven kon toevoegen. Dit was wel een uitdaging want voor elke nieuw duif moest er een nieuw bestand aangemaakt worden met bepaalde code in. Deze code zou er dan voor zorgen dat zijn gegevens geüpload worden op regelmatige tijdstippen in ons dashboard. Zo kon de eindgebruiker zijn duiven gemakkelijk opvolgen en moest hij maar één keer opstaan om te gaan kijken naar zijn duiven.

Systems engineering and analysis

Voor we aan dit project konden beginnen moesten we heel wat voorbereidingen treffen. We hadden enkele gesprekken met de opdrachtgever zodat onze opvatting van de opdracht overeengestemd werd met wat de opdrachtgever verwachte van ons. Ook moesten we onze tijdsindeling bekijken en hoe we onze taken moesten verdelen over het team. Het maken van deze documenten en het voeren van deze gesprekken gebeurden tijdens de lessen van System engineering and analysis.
