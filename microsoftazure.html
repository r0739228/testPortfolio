Microsoft Azure IaC Challenge

Januari 24, 2019 Geel

Intro

Dit is een project dat ik in samenwerking met enkele medestudenten gemaakt heb. Ik kijk zeker enthousiast terug op deze samenwerking met deze fijne mensen. Ook zou ik het in de toekomst zeker open staan om nog enkele projecten te realiseren in samenwerking met deze mensen. Wat ik persoonlijk onthoud aan dit project is de werking van Microsoft Azure. Dit is het eerste project dat ik gemaakt hebt met Microsoft Azure. Het heeft me wel overtuigd om dit in de toekomst ook meer te gebruiken.

Wat is IaC?

Infrastructuur als code is het proces van het beheren en inrichten van computerdatacenters via machineleesbare definitiebestanden, in plaats van fysieke hardwareconfiguratie of interactieve configuratietools.

Opdracht

Het doel van het project was om op een geautomatiseerde manier een website te kunnen hosten. Dit houdt dus concreet in dat wij een oplossing moesten vinden dat er een nieuwe virtuele machine werd aangemaakt met de juiste services al ge√Ønstalleerd. Wij hebben hiervoor gebruik gemaakt van Jenkins. Jenkins monitort het GitHub project en activeerd een heel proces wanneer de klant iets nieuws toevoegt aan zijn GitHub project. Het proces maakt dan een nieuwe virtuele machine aan in Azure en installeerd dan ook automatisch Apache met de gewenste website.

Gebruikte tools

  • Microsoft Azure
    • Microsoft Azure Platform is een Cloud computing-platform van Microsoft waarmee een aantal internetdiensten aangeboden kan worden via het internet of binnen de omgeving van het eigen bedrijf.
  • Jenkins
    • Jenkins is een gratis en open source automatiseringsserver. Het helpt bij het automatiseren van de onderdelen van softwareontwikkeling die verband houden met het bouwen, testen en implementeren, waardoor continue integratie en continue levering mogelijk wordt.
  • GitHub
    • GitHub is een online platform waar software op kan worden geplaatst. GitHub is gebouwd rond het Git-versiebeheersysteem, waardoor GitHub alle mogelijkheden van Git en eigen toevoegingen aanbiedt.
